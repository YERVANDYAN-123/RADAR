# 📡 Проект «Радар» (Radar Project)

Это мой проект, разработанный на платформе **Arduino**.

## 📝 Описание
Проект представляет собой мини-радар. Сервомотор вращает датчик (движения или ультразвуковой), который определяет наличие объектов поблизости. 

## 🛠️ Компоненты (Hardware)
* Плата Arduino (например, Uno / Nano)
* Сервомотор (Servo motor)
* Датчик движения или ультразвуковой датчик (Motion/Ultrasonic sensor)

## 💻 Программное обеспечение
Данные в реальном времени отправляются на компьютер и отображаются в программе Arduino (Монитор порта или специальное приложение).





# 📡 Radar Project

This is my custom project built on the **Arduino** platform.

## 📝 Description
This project is a mini-radar system. A servo motor rotates a sensor (motion or ultrasonic), which detects whether there are any objects nearby.

## 🛠️ Hardware Components
* Arduino board (e.g., Uno / Nano)
* Servo motor
* Motion or Ultrasonic sensor

## 💻 Software
Data is sent to the computer in real-time and displayed within the Arduino software (Serial Monitor or a dedicated application).
